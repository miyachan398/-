◇このアプリについて
　こちらのアプリは、作曲をしたことがないという方に、
　作曲の魅力を知ってもらいたいと思い世策したものです。
　ぜひ、作曲の入門として既存の作曲ソフトの足掛かりとしてお使いください。

　アプリの使用方法については別途ファイルを参照してください。

　こちらのアプリは、Android用とWindows用の2つ用意してあります。
　使用している端末に合ったほうをお選びください。

===================================================================================
◇インストール～アプリケーションの開始～

　▷Androidの場合
　　１．ファイルを解凍してください。

　　２．ファイル内にある『MusicMake.apk』をタップして起動してください。

　　　　※あらかじめ、端末の「設定」→「アプリケーション」→「開発元不明のアプリ」にチェックを入れて
　　　　　有効化しておく必要がある可能性があります。
　　　　　これが無効になっていると、Android Market経由以外からapkファイルを端末にインストールできない恐れがあります。
　　　　　セキュリティの面から、インストールが終わったら「開発元不明のアプリ」のチェックは外しておきますようご注意ください。

　　３．Androidを横に向けて使用になります。

　▷パソコンの場合
　　１．ファイルを解凍してください。

　　２．ファイル内にある『MusicMake_for_PC』を開いていただき、中の『music_make.exe』ファイルを起動してください。

　　３．ウィンドウのサイズは1600×900です。

　　４．ウィンドウを閉じられる際は右上のバツ印を押してください。
===================================================================================
◇Unityで起動したい場合
　▷『Unity_Project』フォルダ内にある『music_make.zip』を解凍して、出てきたファイルをUnityで指定してください。
